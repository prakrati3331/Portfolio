# Prakrati Mishra - Portfolio Website

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS showcasing AI development projects, achievements, and professional experience.

## 🚀 Features

- **Modern Design**: Glassmorphism UI with cosmic theme and smooth animations
- **Responsive Layout**: Mobile-first design that works seamlessly across all devices
- **Interactive Components**: 
  - Animated globe background with orbital effects
  - Image galleries with carousel functionality
  - Smooth scroll animations and transitions
- **Content Sections**:
  - Section with animated background
  - About section
  - Achievements & Hackathons with image galleries
  - Patent Publications & Research
  - Skills showcase
  - Projects portfolio
  - Certifications
  - Experience timeline
  - Contact form

## 🛠️ Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS with custom animations
- **UI Components**: Radix UI + shadcn/ui
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Deployment**: Vercel Analytics

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/prakrati3331/Portfolio.git
cd Portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## 🎨 Customization

### Adding New Achievements

1. Add images to `public/` folder
2. Update `imagePaths` in `components/AchievementGallery.tsx`
3. Add new achievement to `achievements` array in `components/achievements.tsx`

### Updating Theme

Colors and animations are defined in `app/globals.css`:
- CSS variables for colors (lines 7-44)
- Custom animations for orb and stars (lines 79-384)
- Glassmorphism effects (lines 402-453)

## 📸 Image Management

### Achievement Galleries

Images should be placed in the `public/` directory:
- QubitX: `/qubitx-1.jpg`, `/qubitx-2.jpg`, etc.
- Pareto: `/pareto-1.jpg`, `/pareto-2.jpg`, etc.
- PPOC Policy: `/ppoc-1.jpeg`, `/ppoc-2.jpeg`, etc.
- Event Participation: `/a.jpg`, `/b.jpg`, etc.
